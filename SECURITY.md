# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

**Do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via:
1. GitHub's private vulnerability reporting feature (Security tab â†’ Report a vulnerability)
2. Or email directly to the repository owner

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes

## Response Timeline

- **Initial response**: Within 48 hours
- **Status update**: Within 7 days
- **Fix timeline**: Depends on severity

## Scope

This security policy covers:
- The CatchRate Display mod code
- Build and CI/CD pipeline
- Repository secrets and configurations

## What is NOT in scope

- Cobblemon's own security issues (report to Cobblemon team)
- Minecraft client/server vulnerabilities (report to Mojang)
- Third-party dependencies (report to their maintainers)
